# amis-admin-vue3

基于 [amis](https://github.com/baidu/amis) 渲染器，快速搭建自己的管理系统。

此低代码编辑器项目的[原始版本](https://github.com/aisuda/amis-admin)是基于react的,已修改为基于vue3

## 安装依赖
```
npm install
```

### 运行
```
npm run serve
```

### 编译
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
